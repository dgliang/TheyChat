# **TheyChat - Android 即时通讯应用**

## **简介**

**TheyChat** 是一款基于 Android 平台的即时通讯应用，模仿微信的基础聊天功能，支持私聊（单人聊天）和群聊（多人聊天）。通过使用实时通信协议（如 Socket 通信），本应用能够快速、稳定地发送文本消息和图片消息。用户可以通过聊天界面轻松实现与朋友和群体的互动，实时传递信息。
